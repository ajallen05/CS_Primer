# CS Primer
## Description
CS Primer is a collection of programming projects and exercises completed as part of the CS Primer course by OZ Nova. This repository serves as a repository of knowledge, showcasing various concepts and techniques learned throughout the course.

## Features
* Implementation of the Luhn algorithm for validating credit card numbers
* Tic-Tac-Toe game with AI using the Minimax algorithm
* Examples of using Python decorators for functional programming
* Data analysis and processing using Python and CSV files

## Installation
To run the projects in this repository, you will need to have Python installed on your system. You can install the required dependencies by running the following command in your terminal:
```bash
pip install -r Programing/luhns/requirements.txt
pip install -r Programing/tic/requirements.txt
```

## Usage
Each project in the repository has its own usage instructions. Here are some general guidelines:
* Luhn algorithm: Run `python luhns.py` or `python luhns_simple.py` in the `Programing/luhns` directory to test the Luhn algorithm.
* Tic-Tac-Toe game: Run `python main.py` in the `Programing/tic` directory to play the game.
* Reduce examples: Run `python play_reduce.py` in the `Programing/reduce (1)/reduce` directory to see examples of using Python decorators.

## Tech Stack
* Programming language: Python
* Data analysis: CSV files
* Game development: Tic-Tac-Toe game using Python and Minimax algorithm
* Functional programming: Python decorators
* Testing: Unit tests using Python's built-in `unittest` module
